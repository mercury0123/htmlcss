가장 많이 사용하는 데이터 타입을 다루는 메소드

  • 문자열 타입은 index(0부터 시작) 번호를 가지고 있다
  • 문자열 데이터 값 내에서 개행을 하는 것은 추천하는 방법이 아님
  • length 프로퍼티를 이용하면 문자값의 총길이, 마지막 문자 번호는 length-1
  • 문자열 추출 : slice(), substring(), substr()
  • replace(old문자,new문자) : 첫번째 매칭되는 문자만, 대소문자 구분
  • padStart(목표문자길이,채울문자): 앞쪽에 채우기
  • padEnd(목표문자길이,채울문자): 뒤쪽에 채우기
  • charAt(index) : 해당 위치의 문자만 추출
  • indexOf("str"), lastIndexOf("str") : 지정 문자열의 index리턴
  • startWith(), endWith() : 지정문자열이 탐색대상 문자열에서 시작과 끝에 존재하는 지 여부를 체크 return 값 true false 형식으로 반환
