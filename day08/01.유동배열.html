자바스크립트의 배열을 유동배열 : 크기와 데이터의 변경이 가능하다

  • const myArray = ['a','b','c',1,2,3];
  • 자바에서는 데이터 타입이 같아야 하지만, 스크립트 에서는 데이터 타입이 달라도 허용 (유사배열)
  • myArray=[]; 초기값이 없는 빈 배열을 생성 후 데이터를 추가하여 크기를 변경가능
  • 각 데이터들은 (,) 로 분리되고, 자리 index로 관리, 0부터 시작
  • 길이는 역시 length를 이용해서 측정. 크기가 변경될 때 마다 자동으로 체크됨
  • index 번호 순서를 뛰어넘는 등의 변칙적인 사용이 가능. 모두 undefined로 초기화 됨
  • 추후 데이터 통신이나 변환 등에서 자주 사용된다.
