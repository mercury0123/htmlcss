자바스크립트에서 데이터 타입의 확장 개념으로 집합데이터를 다룰 수 있는 객체를 만듦

  • set 안의 중복데이터는 허용하지 않음 : 중복검사 후 존재하는 데이터의 최초의 데이터
  • Set() : 파라미터로는 배열을 주로 사용한다 => 배열의 index와 값을 활용하여 집합데이터를 만듦
  • set의 전용 메소드 들이 존재
  • add : 값을 추가 : name.add(값)
  • size : 크기를 알려줌 : setName.size; o ,size() x <=이런문법이 아님
  • has : 값이 이미 있는지 검사 : setName.has(값) 있으면 true, / false
  • delete : 값을 삭제 : setName.delete(값) 성공하면 true / false 반환
  • clear : 모두 삭제 : setName.clear()
  • forEach(v,i,self(set)) : index == value(set은 키가 별도로 존재하지 않음)
  • const a = new Set([1,2,3]); : 배열 => Set
  • const b = [...a] : Set => 배열로 만들어짐
  • setName.keys(), setName.values(), setName.entries() : 리턴타입이 SetIterator 가 됨 : forEach로 순회하지 x
